How do I access the game panel?

Upload A 64x64 picture in File Manager and rename it to server-icon.png

1. Go the main dashboard of the client area

2. Open https://github.com/Spoony1337 another tab

3. On the client area, copy your username

4. Paste it into the username box on the game panel

5. On the client area, click Regenerate Password & copy the new password

6. Paste it into the password box on the game panel

1. Go to https://freenom.com/ and register an account (you can put fake info)

2. Register the domain you want (has to end in .tk, .gq, .ml, .ga or .cf)

3. Go to https://cloudflare.com/ and register an account

4. Add your domain to Cloudflare

5. Change your domain's nameservers to the new ones (found in freenom's manage domains place)

6. Wait for the domain to be available in cloudflare

7. Create an srv record with this info

- Name: anything

- Protocol: TCP

- Service: _minecraft

- Priority: 1

- Weight: 1

- Port: Your servers port

- Target: Your servers node IP (example: us2.holonodes.host)

- TTL: Auto

8. You should then be able to connect using name-here.yourdomainhere
